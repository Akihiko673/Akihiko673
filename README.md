## Wall: Systems Programming Reimagined

Wall represents a fundamental rethinking of systems-level programming. It is a **strict superset of C11** that preserves C's raw power and transparency while introducing:

### Core Innovations
- **Formal Verification**: Native `prove`, `assume`, and ghost code blocks integrated directly into the type system
- **Middle-Level IR (MIR)**: SSA-based instruction set with semantic opcodes for transactions, races, state machines, and proof obligations
- **Structured Concurrency**: `race` (algorithmic darwinism), `parallel` execution with synchronization, atomic transactions
- **Precision Types**: Exact-width integers (`i7`, `i24`, `i128`) for embedded systems and protocol parsing
- **Module System**: Namespace hierarchies with `derive` for semantic file inclusion—eliminating C's header hell
- **Compact Structs**: Automatic field reordering with native padding elimination
- **Inline Assembly**: Structured operands with full compiler integration

### Technical Architecture
- **Lexer**: Full C11 + Wall keywords with operator duality (`and`, `or`, `not`)
- **Parser**: Recursive descent with Type-First syntax enforcement
- **Semantic Analysis**: Constraint validation and Wall-specific rule enforcement
- **HIR/MIR**: High and Mid-level intermediate representations with type interning
- **VS Code Extension**: Full language support with real-time diagnostics
